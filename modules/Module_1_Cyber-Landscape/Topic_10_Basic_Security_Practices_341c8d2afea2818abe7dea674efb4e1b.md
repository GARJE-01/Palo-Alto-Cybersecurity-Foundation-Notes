# Topic 10: Basic Security Practices

## 🧠 Simple Explanation

**Essential security fundamentals:**

- Strong passwords + MFA
- Firewall + Antivirus
- Regular backups
- Employee training
- Secure network access

---

## 🌍 Real-World Example

- Enable 2FA on email accounts
- Avoid public Wi-Fi for sensitive work
- Regular security awareness training

---

## 💡 Exam & Interview Points

- **Stat: 80% of attacks can be prevented with basic practices**
- Interview question: *"List basic cybersecurity practices"*
- Insight: **Humans are the weakest link**

---

## 🔑 Key Terms

| Term | Definition |
| --- | --- |
| **Firewall** | Blocks unauthorized network access |
| **MFA** | Multi-Factor Authentication (password + something else) |
| **Antivirus** | Detects and removes malicious software |
