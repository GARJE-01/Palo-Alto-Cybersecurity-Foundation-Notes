# Topic 11: Modern Computing Trends

## 🧠 Simple Explanation

Computing is shifting from **traditional systems** to:

- Cloud computing
- Mobile computing
- Work From Home (WFH)
- BYOD (Bring Your Own Device)
- SaaS applications

---

## 🌍 Real-World Example

- Using Google Docs instead of locally installed MS Word
- Working from home on personal laptop
- Accessing company data from phone

---

## ❓ Why It Matters

More flexibility = **Larger attack surface** and complex security

---

## 🔑 Key Terms

| Term | Definition |
| --- | --- |
| **Endpoint** | User device (laptop, phone, tablet) |
| **SaaS** | Software as a Service (accessed via internet) |
| **Attack Surface** | All potential vulnerabilities in a system |

---

## 💡 Exam & Interview Points

- Interview question: *"How modern trends increase attack surface?"*
- Key insight: Decentralized access increases risk

---

## 🚀 Quick Revision

- ▪ Work anywhere = security complexity
- ▪ Multiple device types = harder to manage
- ▪ Attack surface expands significantly
