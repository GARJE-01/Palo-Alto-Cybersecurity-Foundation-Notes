# Topic 19: SaaS Security Best Practices

## 🧠 Simple Explanation

**Securing SaaS applications requires:**

- Multi-Factor Authentication (MFA)
- User behavior monitoring
- Access control (Role-Based Access Control - RBAC)
- Data Loss Prevention (DLP)
- Security awareness training

---

## 🌍 Real-World Example

- Enable 2FA on Gmail/Google Workspace
- Admin controls who can share files
- DLP prevents credit card numbers from being shared

---

## ❓ Why It Matters

Prevents accidental and intentional data leaks.

---

## 🔑 Key Terms

| Term | Definition |
| --- | --- |
| **MFA** | Multi-Factor Authentication (password + device) |
| **RBAC** | Role-Based Access Control (permissions by job role) |
| **DLP** | Data Loss Prevention (prevents data exfiltration) |

---

## 💡 Exam & Interview Points

- Interview question: *"How to secure SaaS applications?"*
- Key practices: MFA, RBAC, monitoring, user training

---

## 🚀 Quick Revision

- ▪ Control access to data
- ▪ Monitor user behavior
- ▪ Train employees on safe practices
