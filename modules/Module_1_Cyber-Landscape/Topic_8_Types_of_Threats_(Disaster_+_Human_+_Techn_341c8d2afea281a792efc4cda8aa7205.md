# Topic 8: Types of Threats (Disaster + Human + Technical)

## 🧠 Simple Explanation

**Threats are not just hacker attacks:**

| Category | Examples |
| --- | --- |
| **Natural Threats** | Flood, fire, earthquake |
| **Technical Threats** | Hardware failure, network outage, software bugs |
| **Human Threats** | Hackers, malicious insiders, accidental deletion |

---

## 🌍 Real-World Example

- Employee accidentally deletes critical database
- Hacker installs malware via email
- Server hardware failure causes data loss

---

## 💡 Exam & Interview Points

- Common mistake: Assuming all threats are malicious
- **Stat: Many incidents are accidental, not intentional**
- Interview question: *"Name all types of threats in cybersecurity"*

---

## 🚀 Quick Revision

- ▪ Threats = Natural + Technical + Human
- ▪ Not all threats are malicious
- ▪ Defense must address all categories
