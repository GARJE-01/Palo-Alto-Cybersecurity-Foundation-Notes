## ✅ Revision Checklist

### Topic 1: What is Malware?

- [ ]  Definition: software designed to harm/exploit/control
- [ ]  Delivery methods: email, websites, USB, etc.

### Topic 2: Types of Malware

- [ ]  Virus: requires host + user action
- [ ]  Worm: self-replicating, fast spread
- [ ]  Trojan: disguised malware
- [ ]  Ransomware: encrypt and demand money
- [ ]  Spyware: steals data
- [ ]  Rootkit: deep system access
- [ ]  Key difference: Virus vs Worm (user action)

- [ ]  Two types: locker and crypto
- [ ]  WannaCry real example
- [ ]  Steps: infection → encryption → ransom
- [ ]  Critical: no guarantee after payment
- [ ]  Prevention: offline backups
- [ ]  3-2-1 backup rule

### Topic 4: Advanced Malware Features

- [ ]  Stealth: hiding techniques
- [ ]  Polymorphism: small code changes
- [ ]  Metamorphism: complete rewrite
- [ ]  Fileless: runs in RAM only
- [ ]  Zero-day: unknown vulnerability
- [ ]  Anti-analysis: detects sandbox
- [ ]  Why traditional AV not enough

## ❓ Key Questions to Answer

1. **What is malware?**
    - Software designed to harm without permission
2. **Why is worm different from virus?**
    - Worm spreads automatically; virus needs user action
3. **What should you do if ransomware hits?**
    - Restore from offline backup, don't pay
4. **What is zero-day?**
    - Vulnerability unknown to vendor, no patch
5. **Why is polymorphism dangerous?**
    - Changes code, defeats signature detection

## 📝 Practice Questions

### Q1: Worm Characteristics

**Which statement about worms is TRUE?**

A) Requires user action to spread

B) Spreads automatically

C) Attaches to files

D) Needs host to survive

✅ **Answer: B**

### Q2: Ransomware Recovery

**Best protection against ransomware?**

A) Pay ransom

B) Offline backups

C) Strong antivirus

D) Encryption

✅ **Answer: B**

### Q3: Advanced Features

**Metamorphic malware differs from polymorphic:**

A) More common

B) Complete code rewrite

C) Less sophisticated

D) Older

✅ **Answer: B**

### Q4: Zero-Day

**Why is zero-day so dangerous?**

A) Spreads fast

B) No patch available

C) Requires root

D) Uses multiple files

✅ **Answer: B**
