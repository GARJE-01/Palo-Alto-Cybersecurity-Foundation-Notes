# Topic 7: Bots & Botnets

> Network of infected devices controlled remotely
> 

---

## 📖 SIMPLE EXPLANATION

👉 **Bot** = infected device (zombie)

👉 **Botnet** = network of many infected devices

**Controlled by attacker using Command & Control (C2) servers**

---

## 🧠 CORE IDEA

### Attacker:

👉 Infects many devices

👉 Controls them together

👉 Uses them for large-scale attacks

### Result:

👉 Massive amplification of attack power

---

## 🌍 REAL-WORLD EXAMPLE

### The Scenario:

**Your laptop infected**

**Millions of other devices infected**

👉 **All controlled together → botnet**

**Attacker commands all at once**

👉 **Massive attack power**

---

## ⚙️ HOW BOTNET WORKS (STEP-BY-STEP)

### Step 1: Malware Infection

- Malware infects device
- Installs bot software

### Step 2: C2 Connection

- Device becomes "bot"
- Connects to Command & Control server
- Waits for commands

### Step 3: Command Reception

- Attacker sends command to C2
- C2 broadcasts to all bots
- Bots receive instructions

### Step 4: Execution

- Bots perform attack
- Simultaneously across network
- Massive scale

### Step 5: Attack Amplification

- Thousands/millions of bots
- Execute same attack
- Overwhelms target

---

## 🔴 TYPES OF BOTNET ATTACKS

### 1️⃣ SPAMMING BOTNET

- **Function**: Send spam emails
- **Scale**: Millions of emails
- **Example**: Rustock botnet
- **Impact**: Email overload

### 2️⃣ DDoS BOTNET

- **Function**: Flood server with traffic
- **Scale**: Gigabits/second
- **Example**: Mirai botnet
- **Impact**: Website unavailable

### 3️⃣ FINANCIAL BOTNET

- **Function**: Steal money/data
- **Scale**: Targeted theft
- **Example**: Banking trojans
- **Impact**: Financial loss

### 4️⃣ CRYPTOMINING BOTNET

- **Function**: Mine cryptocurrency
- **Scale**: CPU power
- **Example**: Crypto miners
- **Impact**: Resource theft

---

## ❓ WHY BOTNETS ARE DANGEROUS

| Aspect | Impact |
| --- | --- |
| 🔊 **Scale** | Large-scale attacks |
| 🔍 **Detection** | Hard to detect |
| 🔄 **Flexibility** | Can adapt and evolve |
| 💰 **Monetization** | Attacker income |
| 🔃 **Persistence** | Long-lasting |

---

## 🔑 KEY TERMS

| Term | Definition |
| --- | --- |
| **Bot** | Infected device |
| **Botnet** | Network of bots |
| **C2 Server** | Control & Communication center |
| **Botmaster** | Attacker controlling botnet |
| **DDoS** | Distributed Denial of Service |
| **Zombie** | Infected computer under control |

---

## ⚠️ IMPORTANT CONCEPT

### To Stop Botnet:

🔥 **Break connection between bots and C2**

- Disrupt C2 servers
- Block bot communications
- Patch vulnerabilities
- Remove bot malware

---

## 📊 FAMOUS BOTNETS

### Mirai (2016)

- IoT-based botnet
- 600,000+ devices
- DDoS attacks
- Dyn attack

### Zeus (2007-2012)

- Banking botnet
- $100 million+ theft
- Credential stealing

### Rustock (2006-2011)

- Spam botnet
- 150+ million messages/day
- Largest spam source
