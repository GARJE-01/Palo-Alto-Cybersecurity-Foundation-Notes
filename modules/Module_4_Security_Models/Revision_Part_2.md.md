## 🎯 Key Questions to Answer

### Q1: Perimeter Security

**Why is perimeter security outdated?**

✅ **Answer:** Modern environments (cloud, remote work) remove clear network boundaries

### Q2: Security Policy

**What are the 5 components?**

✅ **Answer:** Subjects, Objects, Actions, Permissions, Protections

### Q3: DAC vs MAC

**Which allows users to control access?**

✅ **Answer:** DAC (Discretionary Access Control)

### Q4: Trust Management

**What enforces policies?**

✅ **Answer:** Compliance Checker

### Q5: BLP vs Biba

**Which focuses on confidentiality?**

✅ **Answer:** BLP (Bell-La Padula)

## 📝 Practice Questions

### Q1: Perimeter

**Perimeter security fails because:**

A) Too expensive

B) Modern environments blur boundaries

C) Too slow

D) Not secure enough

✅ **Answer: B** - Cloud and remote work removed clear perimeter

### Q2: Security Policy

**In a college system, student accessing result database:**

- Subject: ?
- Object: ?
- Permission: ?

✅ **Answer:** Student, Result database, Read-only

### Q3: DAC vs MAC

**Military system using clearance levels is:**

A) DAC

B) MAC

C) Both

D) Neither

✅ **Answer: B** - MAC (system-controlled, strict)

### Q4: Trust Management

**Compliance checker does what?**

A) Writes policies

B) Enforces rules

C) Stores credentials

D) Logs access

✅ **Answer: B** - Enforces policy compliance

### Q5: BLP vs Biba

**Which prevents reading higher-level data?**

A) BLP

B) Biba

C) Both

D) Neither

✅ **Answer: A** - BLP: No Read Up
