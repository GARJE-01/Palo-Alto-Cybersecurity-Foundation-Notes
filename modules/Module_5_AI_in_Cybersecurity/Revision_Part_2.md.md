## 🎯 KEY QUESTIONS

### Q1: UEBA

**What does UEBA do?**

✅ **Answer:** Monitors user behavior and detects anomalies based on baseline deviations

### Q2: SOAR

**What is SOAR's main advantage?**

✅ **Answer:** Automates security operations, reducing manual work and response time

### Q3: Incident Response

**How does AI improve incident response?**

✅ **Answer:** Faster detection, automatic response, prioritization of threats

### Q4: Metrics

**What is MTTD vs MTTR?**

✅ **Answer:** MTTD=detection time, MTTR=response time; both should be low

### Q5: Data Pipeline

**What is most important step?**

✅ **Answer:** Data cleaning; garbage in, garbage out principle

### Q6: Risks

**What is AI hallucination?**

✅ **Answer:** AI generates incorrect or misleading information confidently

**What does SOAR NOT do?**

A) Orchestrate tools

B) Automate response

C) Detect threats initially

D) Integrate SIEM

**Which AI component reduces MTTR?**

A) Data collection

B) Automated response

D) Manual analysis

✅ **Answer: B**

### Q4: Data Pipeline

A) Reporting

B) Collection

C) Cleaning

D) Storage

✅ **Answer: C** - Bad data ruins everything

### Q5: AI Risks

A) Safe item flagged

B) Alert overload

C) Threat not detected

✅ **Answer: C** - Most dangerous!
