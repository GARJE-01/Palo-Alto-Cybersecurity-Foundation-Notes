# Topic 10: UEBA (User & Entity Behavior Analytics)

> AI-based system that monitors user and entity behavior to detect anomalies
> 

---

## 📖 SIMPLE EXPLANATION

**UEBA** = AI-based behavior monitoring system

What it does:

- Tracks user activity
- Detects abnormal behavior
- Monitors entities

📌 **UEBA identifies activity that deviates from normal behavior**

---

## 🌍 REAL-WORLD EXAMPLE

Scenario:

- Employee logs in from **Mumbai** (normal)
- Suddenly logs in from **USA** at same time (suspicious!)

👉 **UEBA detects this as suspicious** because it violates normal pattern

---

## 🎯 WHY IT MATTERS

✔ **Detect insider threats** - catch employees stealing data

✔ **Detect compromised accounts** - find hacked user accounts

✔ **Works without predefined rules** - catches unknown threats

✔ **AI-based** - learns normal behavior

---

## 🔑 KEY TERMS

**Baseline** = normal behavior pattern

**Anomaly** = unusual activity that deviates from baseline

**Deviation** = how much behavior differs from normal

---

## 🧠 HOW UEBA WORKS

1. **Establish baseline** - learn normal behavior
2. **Monitor activity** - watch user actions
3. **Compare to baseline** - is it normal or abnormal?
4. **Flag anomalies** - alert on suspicious behavior
5. **Investigate** - security team reviews

---

## 👉 UEBA DETECTS

✔ Unusual login times

✔ Suspicious data access

✔ Abnormal locations

✔ Unexpected actions

✔ Lateral movement

✔ Privilege escalation

---
