# Topic 4: Key AI Applications in Security

> Five major ways AI protects cybersecurity
> 

---

## 🔴 1. THREAT DETECTION

**Using patterns to detect attacks**

Examples:

- Malware detection
- Intrusion detection
- Attack pattern recognition

✔ Catches new and known malware

---

## 🔵 2. ANOMALY DETECTION

**Finding unusual behavior**

Examples:

- Unusual login from new location
- Abnormal data access patterns
- Strange network traffic

📌 **AI identifies abnormal patterns indicating attacks**

---

## 🟡 3. PREDICTIVE ANALYSIS

**Predicting future attacks**

Examples:

- Threat intelligence
- Risk prediction
- Vulnerability forecasting

✔ Warns before attacks happen

---

## 🟢 4. AUTOMATED RESPONSE

**Automatically responding to threats**

Examples:

- Block traffic automatically
- Isolate infected system
- Quarantine malware
- Generate alerts

✔ Reduces damage by responding instantly

---

## 🟣 5. BEHAVIOR ANALYSIS (UEBA)

**Monitoring user and entity behavior**

Examples:

- User behavior tracking
- Entity activity monitoring
- Suspicious activity detection

📌 **Detects suspicious activity based on deviations from normal**

---

## 📊 COMPARISON TABLE

| Application | What it does | Example |
| --- | --- | --- |
| Threat Detection | Find attacks | Malware |
| Anomaly Detection | Find unusual behavior | Strange login |
| Predictive Analysis | Predict attacks | Risk scoring |
| Automated Response | React automatically | Block IP |
| Behavior Analysis | Monitor activity | Insider threats |

---

## ⚠️ IMPORTANT CONCEPT

**AI advantages:**

- ✔ Speed
- ✔ Accuracy
- ✔ Automation

**AI limitations:**

- ❌ Needs data
- ❌ Not 100% reliable
- ❌ Requires human oversight
