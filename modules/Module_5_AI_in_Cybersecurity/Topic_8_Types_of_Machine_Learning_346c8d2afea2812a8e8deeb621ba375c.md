# Topic 8: Types of Machine Learning

> 4 main types of ML: Supervised, Unsupervised, Semi-Supervised, Reinforcement
> 

---

## 🔴 1. SUPERVISED LEARNING (SML)

### 📖 SIMPLE EXPLANATION

**Learns using labeled data**

- Input + Output already known

📌 **Uses labeled training data to classify unknown data**

### 🌍 EXAMPLES

- Email → Spam / Not Spam
- File → Malware / Safe
- Login → Legitimate / Fraudulent

### ✔ USE CASES

- Malware detection
- Fraud detection
- Email spam filtering

### ⚠️ LIMITATION

- Needs labeled data
- Requires constant updates

---

## 🔵 2. UNSUPERVISED LEARNING (UML)

### 📖 SIMPLE EXPLANATION

**Learns using unlabeled data**

- Finds patterns on its own
- No pre-labeled examples

📌 **Identifies patterns and anomalies without predefined labels**

### 🌍 EXAMPLES

- Detect unusual login behavior
- Group similar threats
- Find network anomalies

### ✔ USE CASES

- Anomaly detection
- Threat discovery
- Clustering attacks

### ⚠️ LIMITATION

- Less precise than supervised
- Hard to interpret results

---

## 🟡 3. SEMI-SUPERVISED LEARNING

### 📖 SIMPLE EXPLANATION

**Combination of labeled + unlabeled data**

- Uses both types
- Improves accuracy

📌 **Improves accuracy using both types of data**

### ✔ USE CASE

- When limited labeled data available
- Cost reduction
- Better accuracy than unsupervised

---

## 🟢 4. REINFORCEMENT LEARNING (RL)

### 📖 SIMPLE EXPLANATION

**Learns using trial and error**

- Rewards for good actions
- Penalties for bad actions

📌 **Used for adaptive security and real-time decision making**

### 🌍 EXAMPLES

- Adaptive firewall
- Real-time attack response
- Dynamic threat blocking

### ✔ USE CASE

- Dynamic threat response
- Complex attack environments
- Real-time optimization

---

## 📊 COMPARISON TABLE

| Type | Data | Use | Accuracy |
| --- | --- | --- | --- |
| **Supervised** | Labeled | Classification | High |
| **Unsupervised** | Unlabeled | Pattern detection | Medium |
| **Semi-Supervised** | Mixed | Improved accuracy | Medium-High |
| **Reinforcement** | Feedback | Decision making | Improving |

---

## ⚠️ EXAM TRAP

**Supervised vs Unsupervised:**

- Supervised → known labels
- Unsupervised → unknown patterns
