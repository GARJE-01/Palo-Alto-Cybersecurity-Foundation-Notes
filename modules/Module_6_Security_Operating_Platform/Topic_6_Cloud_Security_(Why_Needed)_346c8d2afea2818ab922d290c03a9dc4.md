# Topic 6: Cloud Security (Why Needed)

> Protecting data and systems in the cloud
> 

---

## SIMPLE EXPLANATION

**Cloud Security Challenge** = Data is outside company network

### Modern Organizations Use:

- Google Drive
- AWS
- Azure
- SaaS apps

### The Problem:

- Data outside company control
- No traditional firewall protection
- New attack vectors
- Compliance risks

---

## CLOUD SECURITY CHALLENGES

No Visibility

Data Leakage

Misconfigurations

Unauthorized Access

---

## THE SOLUTION

**Cloud Security Platform**

Monitors cloud activity

Detects threats

Controls access

Prevents data loss

---

## MEMORY TRICK

**"Cloud = No boundary -> Security everywhere"**

## QUICK SUMMARY

Cloud increases risk

Traditional firewalls don't protect cloud

New security needed

Visibility is critical

Control and prevention required
