# Topic 7: Prisma (Platform Overview)

> Platform that secures cloud users and applications
> 

---

## SIMPLE EXPLANATION

**Prisma** = Palo Alto's cloud security platform

### What It Does:

- Protects users (anywhere)
- Protects cloud apps
- Enforces security policies
- Detects threats

---

## TYPES OF PRISMA (IMPORTANT)

### 1. PRISMA ACCESS

- Purpose: Secure remote users
- Provides secure internet access
- Enforces security policies

### 2. PRISMA CLOUD

- Purpose: Secure cloud infrastructure
- Detects misconfigurations
- Identifies threats

---

## HOW PRISMA ACCESS WORKS

User -> Internet -> Prisma -> Application

### What Prisma Does:

1. Inspects user traffic
2. Enforces policies
3. Filters malware
4. Blocks threats
5. Logs activity
